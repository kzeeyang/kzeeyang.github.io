Etcd使用

1月 16, 2019 1536

etcd是一个高可用的分布式键值数据库，可用于服务发现。etcd采用raft一致性算法，基于Go语言实现。

安装

构建集群时修改配置文件initial-cluster参数，本机地址需要一起添加。
新建立集群时initial-cluster-state参数设置为“new”，建立后存在数据需要修改为“existing”。

检测环境

Etcd GO API

etcd本来就是Go开发，官方提供了开发库。需要注意使用的etcd是什么版本，老版本etcd需要使用V2版本的库。
